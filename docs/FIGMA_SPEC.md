# Figma Spec — ACADEMICOS (starter)

This document outlines the basic artboard and tokens for the ACADEMICOS visual system.

Tokens
- Colors: see `design-tokens.json`
- Typography: Space Grotesk (Headings), Inter (Body)

Artboards
- Landing / Hero (1440x1024): central 3D sphere, left content, right visual
- Mission Control (1440x1024): bento layout, floating cards
- Identity Hub profile (1024x768)
